# 北斗店铺助手 — 文件功能说明

> 该项目是一个 Chrome 扩展（Manifest V3），用于从 Temu 平台抓取商品数据、预览并批量导出为 ZIP 压缩包（内含图片和 Excel 清单）。

---

## 一、核心 JS 文件

### 1. `manifest.json` — 扩展配置文件

| 字段 | 值 | 说明 |
|------|-----|------|
| `manifest_version` | 3 | Chrome 扩展 MV3 规范 |
| `name` | 北斗店铺助手 | 扩展名称 |
| `action.default_popup` | `popup.html` | 点击图标打开的弹窗页面 |
| `background.service_worker` | `background.js` | 后台 Service Worker |
| `content_scripts[0].matches` | `*://*.temu.com/*` | 注入到所有 Temu 页面 |
| `content_scripts[0].js` | `content.js` | 注入的内容脚本 |
| `permissions` | activeTab, storage, scripting, downloads | 扩展权限 |
| `host_permissions` | `*://*.temu.com/*`, `*://*.kwcdn.com/*` | 允许访问的域名 |

**功能**：定义扩展的身份、权限、入口文件、以及可以操作哪些网站。

---

### 2. `background.js` — 后台 Service Worker

**运行位置**：Chrome 后台（独立的 Service Worker 线程，页面关闭后仍在运行）

**核心功能**：

| 函数 | 行号 | 功能说明 |
|------|------|----------|
| `downloadImage(url)` | 28-59 | 用 `fetch()` 下载单张图片，设置 Referer/Origin 头，30 秒超时 |
| `downloadImagesConcurrently(products, concurrency, onProgress)` | 61-99 | 并发下载所有商品的图片（默认 5 个 worker） |
| `generateExcelContent(products)` | 101-115 | 生成批量上架用的 Excel 表格（序号、商品名、售价） |
| `handleExportFromPreview(products)` | 182-283 | **完整导出流程**：下载图片 → 建文件夹 → 打包 ZIP → 触发下载 |
| `arrayBufferToBase64(buffer)` | 18-26 | 将 ArrayBuffer 转为 base64，用于 chrome.downloads API |
| `sendExportProgress(current, total, message, completed, success, error)` | 163-180 | 向前端页面发送导出进度消息 |
| `checkProductsExported(productIds)` | 154-161 | 查询商品是否已导出过 |

**消息监听**（第 117-152 行）：
- `exportFromPreview` — 接收来自预览页面的导出请求，触发 `handleExportFromPreview`
- `checkExported` — 查询哪些商品已经导出过（用于预览页面的"已提取"标记）
- `openPopup` — 打开弹窗

**对应页面功能**：
- 用户在前端预览页面点击"导出选中"按钮后，由 background.js 在后台完成所有下载、打包工作
- 最终通过 `chrome.downloads.download()` 下载 ZIP 文件

## 三、第三方依赖库

### 9. `jszip.min.js` — ZIP 打包库

**用途**：在浏览器中创建 ZIP 压缩包。用于将商品图片和 Excel 表格打包成一个 ZIP 文件下载。

**调用位置**：
- `background.js`：`new JSZip()` → `zip.generateAsync({ type: 'blob', compression: 'DEFLATE' })`
- `popup.js`：`new JSZip()` → `zip.generateAsync({ type: 'blob' })`

---

### 10. `FileSaver.min.js` — 文件保存库

**用途**：在浏览器端触发文件下载（`saveAs(blob, filename)`）。

**调用位置**：
- `popup.js`：`saveAs(content, '导出文件_xxx.zip')` — 弹窗导出的最后一步

---

### 11. `xlsx.full.min.js` — Excel 生成库

**用途**：生成 `.xlsx` 格式的 Excel 文件。

**调用位置**：
- `background.js` 和 `popup.js` 中的 `generateExcelContent()` 函数

**生成的 Excel 格式**：

|----------------------------------------|----------------|------|
| 1 | 商品标题... | $10.99 |
| 2 | ... | ... |

## 四、数据流概览

### 场景 A：弹窗获取 → 弹窗导出

```
用户打开弹窗 → 选区域/填参数 → 点"开始获取"
  → popup.js 向 content.js 发消息
  → content.js 在 Temu 页面执行自动翻页 + 提取商品
  → 数据返回 popup.js 的 products 数组
  → 用户点击"导出"
  → popup.js 并发下载图片 → 打包 ZIP → saveAs() 下载
```

### 场景 B：弹窗获取 → 预览导出

```
用户打开弹窗 → 获取商品 → 点"预览"
  → content.js 在页面打开全屏预览浮层
  → 用户勾选商品 → 点"导出选中"
  → content.js 发消息给 background.js
  → background.js 并发下载图片 → 打包 ZIP → chrome.downloads.download() 下载
  → 实时进度通过 content.js 显示在预览浮层上
```

### 导出文件结构

```
导出文件_20260513_120000.zip/
├── 1/
│   ├── 规格图/
│   │   ├── 1-混合色.jpeg
│   │   ├── 2-混合色.jpeg
│   │   └── 3-混合色.jpeg
│   └── 详情图/
│       ├── 1.jpeg
│       ├── 2.jpeg
│       └── 3.jpeg
├── 2/
│   └── ...（同 1 结构）
├── 批量上架附加表格.xlsx
```
